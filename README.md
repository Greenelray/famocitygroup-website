# Famocity Real Estate Website

Premium single-page marketing website for **FAMOCITY REAL ESTATE AND CONSTRUCTION LIMITED** built with:

- Next.js App Router
- Tailwind CSS
- Framer Motion
- Next/Image optimization

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Notes

- Team images are sourced from the provided local assets.
- Property visuals use lightweight SVG illustrations for fast loading.
- Contact form currently posts to a local API route and returns a success response, ready to connect to email or CRM tooling later.
